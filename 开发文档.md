# MC 星空服务器官网开发文档

## 1. 项目概述

- 项目名称：MC 星空服务器官网
- 项目类型：纯静态前端网站
- 线上地址：<https://starrysky.szzz666.top/>
- Minecraft 服务器：`mc.szzz666.top:19132`
- 页面入口：`index.html`
- 技术栈：Vue 3  + Tailwind UI + Axios
- 数据来源：本地静态 JSON、第三方 HTTP API
- 美术风格：简约 圆角 毛玻璃 蓝白

项目没有后端代码、数据库、Node.js 依赖清单或打包流程。部署内容就是仓库内的 HTML、CSS、JavaScript、JSON、图片和音频文件。

## 2. 功能范围

主站当前提供以下功能：

- 展示服务器IP地址端口和实时在线人数
- 使用 `minecraft://` 协议调起 Minecraft 并添加服务器
- 复制服务器 IP 和端口
- 自动、浅色、暗色三种主题模式
- 桌面端和移动端精简显示模式
- 随机背景切换、上一张和背景下载
- 循环播放背景音乐（音乐文件存放在data文件夹中）
- 展示宣传视频和赞助榜（赞助数据存放在data文件夹中）
- 打开聊天、下载、反馈、QQ 频道、QQ群、协管等外部入口
- 提供独立的 Bing 搜索起始页

仓库内不包含以下服务的实现：

- Minecraft 服务器本体
- 在线人数查询服务
- 随机图片服务
- 聊天与协管服务
- 反馈、下载、QQ 频道及 QQ 群页面

这些功能由第三方服务或其他独立部署的服务提供。
